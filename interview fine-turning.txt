Project Name
Interview Chatbot AI

Description
This project is a Data Science and Machine Learning interview chatbot AI, trained on a dataset of questions and answers. It provides a conversational interface to help users prepare for interviews by answering relevant questions.

Features
Provides accurate answers to a variety of interview questions.
Includes sections on data science, machine learning, neural networks, and coding tasks.
Prepares candidates for beginner, intermediate, and advanced-level interviews.
Files
Untitled9.ipynb: Contains the code for the chatbot AI's implementation and logic.
questions_answers.json: A JSON file with a dataset of categorized interview questions and their corresponding answers.
